# kotlin-cli-bazel-sieve-seg

## Description
Find all primes less in a range. Follows functional programming practices.

## Tech stack
- kotlin
- bazel

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
